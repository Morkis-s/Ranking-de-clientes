# Ranking de Clientes

Sistema de cadastro e ranking de clientes desenvolvido em PHP.

## 🚀 Como rodar o projeto

Para executar este projeto na sua máquina, siga os passos abaixo:

### 1. Instalar o XAMPP
É necessário ter o XAMPP instalado no seu computador.

### 2. Colocar o projeto na pasta correta
Copie a pasta do projeto para dentro do diretório:
C:\xampp\htdocs

Exemplo: 
C:\xampp\htdocs\CRUD2

---

### 3. Iniciar os serviços
Abra o XAMPP Control Panel e ligue:

- ✅ Apache  
- ✅ MySQL  

---

### 4. Acessar no navegador

Abra o navegador e digite:
http://localhost/CRUD2


---

## 🛠️ Tecnologias utilizadas

- PHP
- MySQL
- JavaScript
- HTML/CSS

---

## 📌 Observações

- Certifique-se de que o MySQL está rodando corretamente
- Verifique se o banco de dados foi importado (arquivo `.sql`)
